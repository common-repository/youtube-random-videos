Youtube auto feed for wordpress post and pages

== Description ==

This plugin helps you show random youtube videos accroding to kyeword you specify for each post / page, you can also set it's appearance like align, width etc for post / pages.



== Installation ==

* Install the plugin by uploading via wordpress admin.
** OR **
* Upload all plugin files to your plugin directory via ftp
* And activate it from wordpress admin.
